## 1. 系统概述

### 1.1 产品名称
**短视频创作助手**

### 1.2 产品定位
面向短视频创作者（个人博主、企业运营、MCN 机构）的 AI 辅助创作 SaaS 工具。通过接入 DeepSeek 大模型，提供从账号定位、选题规划到脚本制作、爆款分析的全链路创作支持。

### 1.3 目标用户
| 用户类型 | 描述 |
|---------|------|
| 个人创作者 | 希望借助 AI 提升内容产出效率的短视频博主 |
| 企业运营 | 负责企业/品牌短视频账号的运营人员 |
| MCN/工作室 | 管理多账号、需批量生产内容的机构 |
| 系统管理员 | 负责用户管理和系统维护的超级用户 |

### 1.4 核心价值
- 降低内容创作门槛，让非专业人员也能产出高质量脚本和规划
- 提供深度思考（DeepSeek R1）和快速生成（DeepSeek Chat）两种模式，兼顾质量与效率
- 全流程覆盖：定位 → 选题 → 脚本 → 拍摄 → 分析 → 仿写

## 3. 认证与账号管理

### 3.1 登录

**入口**：`/login`

**流程**：
1. 用户输入用户名和密码
2. 前端通过 FingerprintJS 生成设备指纹（`visitorId`）并随表单提交
3. 服务端依次校验：
   - 用户名/密码是否正确（bcrypt 哈希比对）
   - 账号是否启用（`is_active`）
   - 账号是否已过期（`expires_at` 与今日日期比较）
   - 设备指纹是否在允许列表内（设备数量限制）
4. 校验通过后写入 Session，跳转主页

**错误提示**：
- 用户名或密码错误
- 账号已被禁用，请联系管理员
- 账号已过期，请联系管理员续期
- 超出设备数量限制

**交互细节**：
- 密码输入框右侧有 👁 按钮，可切换明文/密文显示
- 登录成功后持续保持 Session，直到主动退出

### 3.2 设备绑定

- 每位普通用户最多绑定 **N 台设备**（由管理员配置），管理员不受限制
- 设备以浏览器指纹（FingerprintJS visitorId）+ IP 地址标识
- 新设备首次登录自动注册；超出上限则拒绝登录并提示

### 3.3 账号有效期

- 普通用户账号设有到期日（`expires_at`，格式 `YYYY-MM-DD`）
- 到期后登录被拒绝，提示联系管理员续期
- 管理员账号无有效期限制

### 3.4 用户资料面板

**触发方式**：
- PC 端：点击侧边栏用户名
- 移动端：点击底部导航「👤 我的」

**展示内容**：
| 字段 | 说明 |
|-----|------|
| 用户名 | 登录账号 |
| 角色 | 普通用户 / 管理员 |
| 账号状态 | 有效 / 已过期（红色高亮） |
| 到期日期 | 具体日期或「永久有效」 |
| 设备数量 | 当前已绑定设备数 |
| 注册日期 | 账号创建时间 |

**操作**：退出登录按钮（清除 Session，跳转登录页）

### 4.1 账号规划 — 定位分析

**功能描述**：根据创作者的行业背景、特长资源和目标平台，AI 输出系统性账号定位方案。

**输入参数**：

| 参数 | 类型 | 说明 |
|-----|------|------|
| 行业领域 | 单选（16 个选项） | 美食/旅行/母婴育儿/健身运动等 |
| 特长优势 | 文本输入 | 创作者的核心竞争力描述 |
| 可用资源 | 多选 | 愿意出镜、专业摄影设备、只有手机、有剪辑技能等 |
| 目标平台 | 多选 | 抖音、快手、小红书、视频号、B站 |
| 账号类型 | 多选 | 个人IP、企业账号、品牌账号、机构账号 |
| 内容风格 | 多选 | 干货知识、搞笑娱乐、生活记录等（20 个选项） |
| 内容形式 | 多选 | 口播、情景剧、教程、Vlog、测评等（12 个选项） |

**AI 输出内容**：
1. **账号定位**：一句话方向（20 字以内）
2. **内容方向**：3-5 个具体方向，每个附选题示例
3. **差异化优势**：与同赛道竞争者的独特之处
4. **目标人群**：受众画像
5. **起步建议**：冷启动的 3 条实操建议
6. **发展建议**：第 1/2-3/4-6 个月的阶段性目标

**生成模式**：快速生成（deepseek-chat）/ 深度思考（deepseek-reasoner）

**联动**：定位分析结果可作为「内容规划」的参考输入（Step 1 → Step 2）

---

### 4.2 账号规划 — 内容规划

**功能描述**：基于账号定位，AI 生成 30 天内容发布排期计划，以可视化日历和图表呈现。

**输入参数**：

| 参数 | 类型 | 说明 |
|-----|------|------|
| 行业领域 | 文本/选择 | 同定位分析 |
| 目标平台 | 文本输入 | 主要发布平台 |
| 当前粉丝量 | 单选 | 0-1千 / 1千-1万 / 1万-10万 / 10万以上 |
| 每天可用时间 | 单选 | <1小时 / 1-2小时 / 2-4小时 / 4小时以上 |
| 定位分析参考 | 自动填入 | Step 1 完成后自动带入 |
| 目标客户画像 | 文本输入（选填） | 补充受众描述 |

**AI 输出（JSON 结构化）**：
- 每日发布数量
- 最佳发布时间（2-3 个时间点）
- 内容类型分布（30 天各类型天数占比）
- 每周内容模板（周一到周日的具体选题）
- 成长建议（针对粉丝阶段）
- 决策依据（发布时间/内容配比/周节奏的解释）

**可视化展示**：
- 内容类型分布饼图
- 最佳发布时间轴
- 一周内容排期表格
- 30 天内容日历（颜色标注不同内容类型）

**生成模式**：快速生成 / 深度思考

### 4.3 爆款选题

**功能描述**：根据赛道和平台，AI 生成 10 个具有传播潜力的爆款选题，并说明每个选题的爆款理由。

**输入参数**：

| 参数 | 类型 | 说明 |
|-----|------|------|
| 赛道领域 | 文本输入（必填） | 内容方向 |
| 创作者特长 | 文本输入（选填） | 与定位联动 |
| 发布平台 | 单选 | 抖音/快手/小红书/视频号/B站 |
| 内容方向偏好 | 文本输入（选填） | 进一步限定方向 |
| 爆款元素偏好 | 多选 Chip | 悬念/反转/挑战/共鸣/干货等 |

**AI 输出格式**：
```
1. 【标题】xxx
   【爆款理由】xxx
（共 10 条）
```

**联动**：每条选题右侧提供「用于脚本制作」按钮，点击自动填入脚本制作的视频主题并跳转

**生成模式**：快速生成 / 深度思考

### 4.4 变现选题

**功能描述**：根据账号赛道和粉丝量级，AI 推荐最适合的变现路径，并提供对应的内容选题方向。

**输入参数**：

| 参数 | 类型 | 说明 |
|-----|------|------|
| 赛道领域 | 文本输入（必填） | |
| 当前粉丝量 | 单选 | 0-1千 / 1千-1万 / 1万-10万 / 10万以上 |
| 创作者特长 | 文本输入（选填） | |
| 变现方式偏好 | 文本输入（选填） | 接广告/带货/知识付费等 |
| 内容调性偏好 | 多选 Chip | 专业权威/亲切接地气/幽默搞笑等 |

**AI 输出内容**：
1. **最适合的变现方式**：2-3 种变现路径，附理由
2. **变现选题方向**：每种变现方式对应 3 个具体选题，说明如何自然植入变现点
3. **操作路径**：分 3 步走的变现计划

**生成模式**：快速生成 / 深度思考

### 4.5 脚本制作

**功能描述**：根据视频主题和风格设置，AI 生成包含台词、镜头说明、时间节点和话题标签的完整拍摄脚本。

**输入参数**：

| 参数 | 类型 | 默认值 | 说明 |
|-----|------|--------|------|
| 视频主题 | 文本输入（必填） | — | 支持历史输入下拉提示（localStorage，最多 15 条） |
| 视频类型 | 下拉选择 | 通用 | 美食探店/旅行/知识教程/种草/搞笑/励志 |
| 文案风格 | 下拉选择 | 幽默 | 幽默/专业/种草/悬疑 |
| 视频时长 | 数字输入 | 60 | 支持直接输入秒数，也可从预设选项选择（15/30/60/90/120/180/300 秒） |

**AI 输出结构**：
- 开场钩子（前 3 秒）
- 3-4 个内容节点（台词 + 镜头说明 + 景别/动作）
- 时间节点标注（秒）
- 结尾互动引导
- 5 个话题标签

**历史记录**：每次生成时，视频主题自动保存至 localStorage（最多 15 条，去重），下次输入时可从历史下拉选择。

**生成模式**：快速生成 / 深度思考

### 4.6 分镜拍摄表

**功能描述**：基于已生成的脚本，一键生成专业分镜拍摄表，辅助 AI 绘图和 AI 视频工具使用。

**触发方式**：脚本制作完成后，点击「📋 生成分镜拍摄表」按钮（紫色主按钮，全宽显示）

**输出格式**：Markdown 表格，包含以下 8 列：

| 列名 | 说明 |
|-----|------|
| 镜头号 | 顺序编号 |
| 景别 | 特写/近景/中景/远景/大远景 |
| 画面描述 | 场景、人物动作、气氛的可视化描述 |
| 文生图提示词 | 中文描述 + `EN:` 开头的英文提示词（供 Midjourney/Stable Diffusion 使用） |
| 图生视频提示词 | 中文描述 + `EN:` 开头的英文提示词（供 Runway/Kling 使用） |
| 时长(秒) | 该镜头建议时长 |
| 旁白/对白 | 对应台词或画外音 |
| 备注 | 拍摄注意事项、特效说明 |

**提示词说明**：
- 文生图/图生视频列同时提供中文（理解用）和英文（工具使用）
- 页面顶部显示黄色提示横幅：建议复制 `EN:` 开头的英文部分粘贴到工具

**生成模式**：继承脚本制作当前选中的模型（快速/深度思考）

### 4.7 文案二创

**功能描述**：将用户粘贴的原始文案改写为 3 个不同风格的版本，用于内容差异化发布或二次创作。

**输入参数**：

| 参数 | 类型 | 说明 |
|-----|------|------|
| 原始文案 | 多行文本（必填） | 粘贴需要改写的文案或视频标题 |

**AI 输出内容**：
- **版本 1 — 悬疑反转风**：设置悬念、制造反转
- **版本 2 — 干货清单风**：「X 个技巧/方法/原因」结构
- **版本 3 — 情感共鸣风**：从情感角度切入，引发共鸣

每个版本包含：开场钩子 + 正文 + 互动引导

**生成模式**：快速生成 / 深度思考

### 4.8 爆款拆解

**功能描述**：对爆款短视频进行深度分析，拆解其传播逻辑和内容公式，提炼可复用方法论。

**两种输入方式**：

**方式一：粘贴视频链接**
- 支持平台：B站、YouTube、抖音（分享文案中的链接）
- 系统自动抓取视频信息（标题、文案、数据）
- B站/YouTube 提供「预加载」功能，抓取完成后可查看详情再决定是否分析

**方式二：搜索爆款视频**
- 输入关键词，同时搜索 B站、YouTube、视频号
- 三个平台结果交叉排序展示
- 每条结果显示：标题、平台、发布者、简介
- 点击结果仅填入标题字段，用户可手动补充内容后再生成（不自动触发生成）

**AI 输出分析维度**：
1. **钩子分析**：前 3 秒如何抓住注意力，使用了哪些钩子技巧
2. **内容结构**：整体叙事结构（起承转合/问题-解决/故事弧线等）
3. **情绪节奏**：情绪高潮节点分布，留存策略
4. **传播因子**：引发转发/评论的核心驱动力
5. **爆款公式**：提炼 1 个可复用公式，附套用示例

**搜索来源**：B站 API、YouTube Data API v3、视频号 H5 页面抓取

**生成模式**：快速生成 / 深度思考

### 4.9 爆款仿写

**功能描述**：提取参考爆款视频的内容公式（钩子/结构/节奏/传播因子），并将其迁移应用到用户指定的全新话题上，生成原创脚本。

**输入参数**：

| 参数 | 类型 | 说明 |
|-----|------|------|
| 参考视频标题 | 文本输入（必填） | 爆款视频的标题 |
| 参考视频文案 | 多行文本（选填） | 视频内容/脚本，提供后分析更精准 |
| 我的话题 | 文本输入（必填） | 要创作的新视频主题 |
| 视频风格 | 下拉选择 | 幽默/专业/种草/悬疑 |
| 目标时长 | 数字输入 | 约多少秒 |

**AI 输出结构**：

**第一部分：公式提炼**
- 钩子类型
- 内容结构
- 情绪节奏
- 传播因子

**第二部分：仿写脚本**
- 按提炼的公式为新话题创作完整脚本
- 每段标注时间节点和镜头说明
- 结尾互动引导
- 附 5 个话题标签

**原创性说明**：内容须完全原创，不得抄袭参考视频具体文案（神形分离原则）

**生成模式**：快速生成 / 深度思考

### 4.10 编导专栏

**功能描述**：根据视频主题、拍摄场景和可用设备，AI 输出完整的编导拍摄方案，适合无拍摄经验的新手执行。

**输入参数**：

| 参数 | 类型 | 说明 |
|-----|------|------|
| 视频主题 | 文本输入（必填） | |
| 拍摄场景 | 单选 | 室内家居/摄影棚/户外自然/户外街道/餐厅咖啡厅/办公室/其他 |
| 可用设备 | 多选 | 仅手机/手机+稳定器/单反微单/专业摄像机/无人机/GoPro |

**AI 输出内容**：
1. **拍摄思路**：整体拍摄逻辑和视觉风格建议
2. **分镜方案**：4-6 个镜头，每个镜头包含景别、拍摄角度、运镜方式、画面内容
3. **灯光建议**：根据场景和设备给出可行的补光方案
4. **剪辑节奏**：建议的剪辑节奏和转场风格
5. **拍摄技巧**：2-3 条针对该题材的实用技巧

**生成模式**：快速生成 / 深度思考

## 5. AI 生成引擎

### 5.1 模型选择

| 模式 | 模型 ID | 特点 | 适用场景 |
|-----|---------|------|---------|
| ⚡ 快速生成 | `deepseek-chat` | 响应快（1-3秒出内容），质量稳定 | 日常创作、快速迭代 |
| ✨ 深度思考 | `deepseek-reasoner` | 深度推理（思考阶段 20-40 秒），质量更高 | 复杂定位分析、高质量脚本 |

### 5.2 流式输出

所有 AI 接口均采用 HTTP 流式响应（`text/plain; charset=utf-8`），前端逐块读取并实时渲染 Markdown，避免长时间等待白屏。

### 5.3 等待体验优化

**深度思考模式**在思考阶段（20-40 秒）显示动画进度条：
- 🌊 渐变光泽进度条（紫色，42 秒填满动画）
- 提示文字：「✨ 正在深度思考中，请稍候...」
- 思考阶段结束、内容开始输出时，进度条自动消失，内容实时更新

### 5.4 结果存储

- 每个功能模块的生成结果存储在内存（`resultStore`）和 `sessionStorage` 中
- 支持在快速/深度思考两个 Tab 间切换查看不同模型的结果
- 有结果的 Tab 显示蓝色小圆点标记

### 5.5 Admin Prompt Debug

管理员账号生成内容后，结果区域显示「ADMIN Prompt Debug」折叠面板，展示：
- **SYSTEM PROMPT**：系统角色设定
- **USER PROMPT**：本次生成的完整用户提示词

方便管理员审查和调优 Prompt 质量，对普通用户不可见。

### 10.3 缺失功能清单与优先级

优先级定义：
- 🔴 **P0 — 高优先级**：用户高频需求、实现成本低、直接提升使用价值
- 🟡 **P1 — 中优先级**：有明确需求但实现复杂度中等
- 🟢 **P2 — 低优先级**：锦上添花，暂不影响核心体验

---

#### 🔴 P0 — 立即补充

**① 标题优化工具**

- **描述**：输入一个视频主题或草稿标题，AI 一次生成 10 个不同风格的标题变体（悬念型、数字型、共鸣型、问句型、干货型等），供 A/B 测试选用
- **价值**：标题是短视频最核心的流量入口，直接影响完播率和推荐量；用户写标题是最高频的日常需求
- **实现成本**：低，新增一个 AI prompt + 简单 UI 即可
- **建议入口**：独立功能模块，或在脚本制作结果下方增加「生成标题变体」按钮

---

**② 发布文案 + 话题标签（平台专项优化）**

- **现状说明**：脚本制作已在脚本末尾自动生成 5 个话题标签；视频主题输入框中用户填写的标题本身可用作发布文案。基础能力已具备。
- **缺口**：现有标签为通用建议，未针对各平台规则区分（小红书标签策略、抖音话题关联热榜、视频号话题入口不同）；发布正文文案（尤其小红书需要有格式感的种草文案）未单独提供
- **建议补充**：在脚本结果下方增加「生成平台发布包」按钮，输出：分平台专属发布文案 + 对应平台优化标签组合
- **实现成本**：低

---

**③ 热点追踪 / 实时热榜**

- **与爆款拆解的关系**：爆款拆解已有关键词搜索功能（覆盖 B站/YouTube/视频号），用户主动搜索即可发现热门视频并分析——这在一定程度上覆盖了「热点发现」的需求。两者的核心差异在于：爆款拆解是「你去找热点」，热点追踪是「热点主动来找你」。
- **真正缺失的**：平台级实时热榜的主动推送（微博热搜、抖音热点榜、今日头条热榜），用户无需搜索就能看到当前全网在讨论什么
- **价值**：蹭热点是短视频涨粉效率最高的方法之一；但鉴于爆款拆解已部分覆盖，优先级可从 P0 下调至 P1
- **实现成本**：中（需接入第三方热榜 API，有维护成本）

---

**④ 封面文案建议**

- **描述**：根据视频主题，AI 生成 5-8 个适合作为封面大字的短句（通常 4-10 字），并说明每条的吸引力原理
- **价值**：封面是短视频的「商品包装」，封面文字直接影响点击率；现有系统完全缺失
- **实现成本**：极低，一个简单 prompt 即可
- **建议入口**：脚本制作结果区域，与「生成分镜拍摄表」并列

---

#### 🟡 P1 — 计划补充

**⑤ 长视频 / 图文转短视频脚本**

- **描述**：用户粘贴一篇文章、公众号推文、或长视频文字稿，AI 将其压缩改写为适合短视频平台的 60-180 秒脚本，保留核心观点
- **价值**：大量创作者有存量图文内容需要「视频化」，这是高频转化需求
- **实现成本**：中（需处理长文本输入，可能需要分段处理）
- **建议入口**：文案二创模块下增加「长文转视频脚本」子功能

---

**⑥ 账号健康诊断**

- **描述**：用户输入账号近期数据（粉丝增减、平均播放量、完播率、互动率等），AI 给出问题诊断和改进建议（如：完播率低说明开头不够吸引、互动率低说明缺乏引导等）
- **价值**：帮助用户发现内容问题，具备决策支持价值，是高价值的咨询型功能
- **实现成本**：中（需设计数据输入表单 + 诊断 prompt）

---

**⑦ 竞品账号分析**

- **描述**：用户输入竞品账号名称或近期爆款标题列表，AI 分析其内容策略、选题规律、发布频率规律，输出可借鉴的竞品策略报告
- **价值**：竞品研究是策划阶段的重要环节，帮助用户找到差异化空间
- **实现成本**：中（无法直接调 API，需用户手动输入竞品信息）

---

**⑧ 跨平台内容适配**

- **描述**：将一份内容（脚本/文案）自动适配为不同平台版本：抖音版（口语化、节奏快）、小红书版（种草感、emoji 丰富）、视频号版（偏正式、适合中年用户）、B站版（可更长、更有深度）
- **价值**：多平台运营是趋势，手动改写耗时；一键适配节省大量时间
- **实现成本**：中（需对各平台特点深度 prompt 调教）

---

#### 🟢 P2 — 未来迭代

**⑨ 选题库 / 收藏夹**

- **描述**：用户可将生成的选题、脚本、分析结果收藏保存，形成个人选题库，支持分类和搜索
- **价值**：避免好内容「生成即遗忘」，提升工具的长期留存价值
- **实现成本**：高（需数据库设计 + 前端交互）

---

**⑩ BGM 推荐**

- **描述**：根据视频类型和风格，推荐适合的背景音乐类型（如：美食类 → 轻快爵士；励志类 → 史诗感管弦乐），附推荐理由和搜索关键词（在各平台音乐库中搜索）
- **价值**：BGM 影响视频情绪，但用户普遍不知道从何选起
- **实现成本**：低（纯文字建议，无需接入音乐 API）

---

**⑪ 评论区运营助手**

- **描述**：用户粘贴评论内容，AI 生成得体、有互动性的回复文案（区分正面评论、疑问、负面评论等场景）
- **价值**：评论互动率影响账号权重，创作者通常忽视评论运营
- **实现成本**：低

---

**⑫ 内容日历可视化管理**

- **描述**：将「内容规划」生成的 30 天排期与真实日历结合，支持用户标记「已拍摄」「已发布」「待制作」状态，形成可管理的内容日历
- **价值**：将 AI 建议落地为可执行的工作流
- **实现成本**：高（需数据库 + 交互复杂）
